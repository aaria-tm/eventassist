# eventassist
EventFlow is a modern event planning web application designed to simplify vendor booking, budget management, and event coordination. It provides an interactive and visually rich experience tailored for different types of events like weddings, birthdays, engagements, and more.

---

## Overview

EventFlow helps users plan events efficiently by allowing them to:

* Browse and compare vendors
* Calculate costs dynamically
* Manage budgets intelligently
* Handle last-minute vendor replacements
* Customize event timelines
* Explore live artists and entertainment options

The platform adapts its UI and experience based on event type, making it feel personalized and immersive.

---

## Features

### 1. Dynamic Event Themes

* Wedding-based UI: red, gold, brown, white palette
* Party-based UI: blue, black, beige, silver palette
* Automatically changes layout and styling based on selected event

---

### 2. Location-Based Vendor System

Supports multiple cities:

* Mumbai
* Delhi
* Pune
* Bangalore

Each location can have different vendors for:

* Catering
* Cakes
* Decor
* Entertainment
* Entry services

---

### 3. Catering System (Per Plate Logic)

* Displays:

  * Price per plate
  * Menu items included
* User selects number of plates
* Total cost calculated dynamically

---

### 4. Cake Selection (Per KG)

* Multiple cake vendors available
* User selects required weight (kg)
* Automatic price calculation

---

### 5. Vendor Comparison System

* Select multiple vendors
* Click “Compare”
* Displays side-by-side comparison:

  * Price
  * Services
  * Value for money

---

### 6. Budget Management

* User sets total budget
* System tracks expenses
* If budget exceeds:

  * Suggests cheaper alternatives
  * Recommends cost reduction areas (e.g., catering, decor, cake)

---

### 7. Smart Timeline Generator

* User inputs event time
* Timeline auto-adjusts:

  * Setup time
  * Vendor arrival
  * Ceremony flow
  * Cleanup

---

### 8. Last-Minute Vendor Support

Users can request emergency replacements for:

* Catering
* Decoration
* Cake
* Entry
* Entertainment

---

### 9. Live Artists & Entertainment

Includes:

* Live singers
* Dhol performers
* DJs
* Live painters
* Floral artists

---

### 10. Multi-Event Support

Supports:

* Wedding
* Engagement
* Birthday
* Baby Shower
* Naming Ceremony
* Cocktail Party

Each event type has customized UI/UX and service flow.
